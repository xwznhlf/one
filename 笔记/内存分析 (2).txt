 
   1、 每一个程序要想运行起来，需要加载到内存中， 
         内存给每个程序分配的空间有限，  这个空间又分为好多区域
       
          缓冲区 ， 栈内存， 堆内存， 常量池 ，静态域  

          栈内存 一般用来存放   变量名， 变量值  与 对象名   又先进后出的规则   int age =18 ;

          堆内存， 只要是 new 的一个类， 就是在创建对象， 就会把这个对象， 放到 堆内存中，每一个对象会消耗一个堆内存区域

                   会生成一个对应的 十六进制的地址，     对象名在栈内存中， 然后 对象名所引用的就是 对象的地址， 
 			
                   所以说， 引用数据类型就是，引用堆内存的地址， 比如   类（对象） ， 数组 ， String  ,
       	
                   这些东西创建偶都需要 new 。  

                   String  nanem = new String("张三")； 简写  String name = "张三" ；

                   堆内存中的空间一旦被 创建出来了， 就不会被 清除 。


    2 、程序-- 》硬盘（ROM  存储内存）      运行--》运行内存（RAM） 



    3 、数组属引用类型，数组型数据是对象(object)，数组中的每个元素相当于该对象的成员变量
